<h3><b>Description：</b></h3>
基于Java17+SpringBoot+RuoYi+Playwright+Redis+WebSocket+Nginx+Vue+微信小程序的多 AI 集成与自动化执行项目。
<h3><b>功能模块：</b></h3>
AI主机核心服务端、控制台后端、控制台前端、控制台微信小程序小程序端
<h3><b>效果展示：</b></h3>
<h3><b>Web端：</b></h3>
首页统一管理多AI登录状态，检查用户登录状态，显示用户名。
